Highlight #1 Version Control Systems

A version control system is a system that helps people keep an organized record of revised versions of files. It keeps a record of who chnaged what, when they chnaged it, whats different about it, and it even merge chnages to the same file. It holds a common repository that holds all the latest files that everyone who has authority can access. It is a very helpful tool for lage and small companies, you can never be to careful

Highlight #2 What is Git

Git is a free version control system. Its been designed to work with large and small projects with speed and efficiency. It has three basic steps to copy something over to the remote repository, first add, second commit, third push. With these three basic steps you would be able to share your work with anyone else who has access to the repositroy.

Highlight #3 Why Use Git

The reason a company would opt to use git because not only is it free but every user would have a complete copy of the repository data stored locally on their computer. This means that anyone wouldn't have to have internet access and could work offline. Because of this the processes being excuted will run much faster. Also Git will prevent any unintentional overwrites of a file on the repository, meaning no one can delete your saved work or any unforseen crashes your work will still be there.
